Facade

RoboTamer Facade is a component dependency substitution system to achieve component independentcy via the facade programming pattern.

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by robotamer

Facade's main focus is a to be a component dependency substitution system, and will achieve component interdependency via the facade programming pattern.

With Facade you can build a cli or web framework by mixing components from Pear, Zend, Composer or just standalone classes from PHPClasses.

Facade also includes a collection of libraries, and recommends tested components to help you get started.
